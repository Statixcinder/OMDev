Matrikon OPC Explorer is a test client that uses the regular OPC interface. The latest version can be found in
http://www.matrikonopc.com/downloads/

UA Expert is a test client that uses the OPC UA interface. The latest version can be found in
http://www.unified-automation.com/opc-ua-clients/

Simple OPC Client is a simple test client that utilizes both the regular OPC interface and the Simulation control (SC) interface in OPCKit. This test client makes possible to control the simulation (start, stop, step), as the simulation starts in the stopped state when OPC (or OPC UA) is linked.
